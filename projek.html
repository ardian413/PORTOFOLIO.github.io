Website Organisasi Mahasiswa

Proyek Desktop | Dibuat dengan HTML, CSS, JavaScript, PHP, Dan Database

Deskripsi Proyek

Website Organisasi ini dikembangkan sebagai platform digital resmi untuk memperkenalkan visi, misi, program kerja, dan kegiatan dari suatu organisasi kepada publik secara lebih profesional dan terstruktur. Website ini bertujuan untuk meningkatkan transparansi, komunikasi, serta keterlibatan anggota dan masyarakat luas terhadap aktivitas organisasi.

Fitur Unggulan:

  • 🏛️ Profil lengkap organisasi (visi, misi, sejarah, struktur)
  • 📆 Kalender dan dokumentasi kegiatan organisasi
  • 📰 Halaman berita atau pengumuman terbaru
  • 🖼️ Galeri foto kegiatan
  • 🌐 Desain responsif dan ringan untuk berbagai perangkatn

Aplikasi Manajemen Jadwal Mahasiswa

Proyek Desktop | Dibuat dengan Python, Tkinter, SQLite

Deskripsi Proyek

Aplikasi ini dirancang untuk membantu mahasiswa mengelola jadwal kuliah dan tugas mereka secara efisien. Dengan antarmuka yang ramah pengguna, pengguna dapat menambahkan, mengedit, menghapus, serta melihat pengingat tugas dan jadwal per hari.

Fitur Unggulan:

  • CRUD Jadwal dan Tugas
  • Reminder Otomatis
  • Kalender Visual (tkcalendar)
  • Statistik dan Timer Belajar (Pomodoro)
